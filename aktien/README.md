# Aktien Modul 📉💼📊

Dieses Modul zeigt Depot- und Aktienfunktionen im zentralen FinanzApp-Server 🚀

## Einstieg 🎯
- URL: `/aktien/`
- Datei: `ShareView.html`
- Assets: `css/ShareView.css`, `js/ShareView.js`

## Starten ▶️
Vom Projekt-Root:
```bash
npm run backend:start
```
